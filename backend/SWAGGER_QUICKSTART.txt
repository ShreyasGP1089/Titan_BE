🎯 HOW TO USE SWAGGER UI
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open: http://localhost:5000/docs

2. Find an endpoint (e.g., "POST /api/v1/shopping/search")

3. Click to expand it

4. Click "Try it out" button

5. Fill in the request body:
   {
     "query": "yoga mat",
     "price_limit": 2000,
     "limit": 5
   }

6. Click "Execute"

7. See the response!
