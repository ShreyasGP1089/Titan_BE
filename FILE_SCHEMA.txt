## Data Flow Architecture

```
┌─────────────────────────────────────────────────────────────┐
│ USER                                                         │
│ "I want to start hiking under 10000"                        │
└────────────────────────┬────────────────────────────────────┘
                         │
                         │ POST /api/v1/query
                         │ {"query": "..."}
                         ▼
┌─────────────────────────────────────────────────────────────┐
│ backend/api_swagger.py (Port 5000)                          │
│ ├─ Authenticate API key                                     │
│ ├─ LocalModelClient.parse(query)                           │
│ └─ Route by intent                                          │
└────────────────────────┬────────────────────────────────────┘
                         │
                         │ POST /parse-query
                         │ {"query": "..."}
                         ▼
┌─────────────────────────────────────────────────────────────┐
│ training/local_model_server.py (Port 8000)                  │
│ ├─ Build ChatML prompt                                      │
│ ├─ Generate with Qwen3-4B + LoRA                           │
│ └─ Return structured JSON                                   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         │ {"intent":"task","arguments":{...}}
                         ▼
┌─────────────────────────────────────────────────────────────┐
│ backend/tools/task_tool.py                                  │
│ ├─ Search products for each item                           │
│ └─ Call BudgetOptimizer                                     │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│ backend/tools/budget_optimizer.py                           │
│ ├─ Score all products                                       │
│ ├─ Select mandatory items (with lookahead)                 │
│ ├─ Add optional items (by utility)                         │
│ └─ Return optimized kit                                     │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│ backend/services/hybrid_search.py                           │
│ ├─ SQL filtering (sport, category, price)                  │
│ ├─ Semantic search (pgvector)                              │
│ └─ Return ranked products                                   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│ PostgreSQL + pgvector                                        │
│ ├─ products table                                           │
│ └─ product_embeddings table                                 │
└────────────────────────┬────────────────────────────────────┘
                         │
                         │ Results
                         ▼
┌─────────────────────────────────────────────────────────────┐
│ Response to User                                             │
│ {                                                            │
│   "activity": "Hiking",                                      │
│   "total_cost": 9500,                                        │
│   "items": [...]                                             │
│ }                                                            │
└─────────────────────────────────────────────────────────────┘
```
