================================================================================
PERFORMANCE IMPACT
================================================================================

BEFORE (HTTP-based embeddings):
   - Network overhead: ~10-50ms per request
   - Serialization: JSON encoding/decoding
   - Model server load: handles all embeddings
   - Single point of failure: model server down = no embeddings

AFTER (Local embeddings):
   - Network overhead: 0ms (local)
   - Serialization: None
   - Backend handles: embeddings independently
   - Resilience: embeddings work even if model server down

Expected improvements:
   ✓ Faster embedding generation (~50% reduction in latency)
   ✓ Better scalability (parallel embedding generation)
   ✓ Simpler architecture (fewer HTTP dependencies)
   ✓ Easier debugging (local = predictable)
